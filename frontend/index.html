Built With Science

End-to-end pipeline from raw NASA data to live AI predictions

📡

Data Engineering

NASA Exoplanet Archive data loaded, 10 key features selected (planetary + stellar), duplicates and outliers removed, missing values imputed via median/mode, clipped at 1st–99th percentile.

pandas numpy NASA Archive
⚗️

Feature Engineering

Derived habitability index (temperature, radius, distance scores), stellar compatibility index, and orbital stability factor. Applied StandardScaler normalisation. One-Hot Encoded 291 spectral types → 304 total features.

OHE StandardScaler Feature Engineering
🤖

Model Training

Three models compared: Logistic Regression, Random Forest, XGBoost. XGBoost selected as best model. Hyperparameter search via RandomizedSearchCV (10 iterations, 3-fold CV). Leakage features removed before final training.

XGBoost RandomizedSearchCV scikit-learn
🔌

Backend API

Flask REST API exposing /api/predict (single + batch), /api/rank (top planets), /api/sample (auto-fill), and /api/health. utils.py reconstructs the exact 304-feature vector automatically from 9 user inputs.

Flask REST API joblib
🎨

Frontend Engineering

Space-themed glassmorphism UI built with Bootstrap 5. Animated starfield, orbital loading rings, SVG habitability arc, batch CSV/JSON upload, live rankings table, and curated sample picker.

Bootstrap 5 Glassmorphism Vanilla JS
